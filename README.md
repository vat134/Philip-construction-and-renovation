# Philip-construction-and-renovation
Website about construction work.

My project in which I created a static website to attract customers and showcase the master's work. The website does not have any particular logic and was created based on the client's preferences and taste.

Simple and elegant design with animations and features such as:
- Navigation buttons that scroll the page to the desired block. 
- Visible buttons to open other pages with projects.
- Photo filters by category and the ability to open them for closer inspection.
- Clicking on a phone number or email address immediately opens the email or dials the phone number for quick communication.

The website itself:

https://philip-construction-and-renovation.netlify.app/
